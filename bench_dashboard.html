📝 Methodology

All benchmarks run in release mode with identical data generation seeds. Timing uses std::time::Instant with 2 warmup iterations discarded, then averaged over multiple timed iterations with ±σ (standard deviation) reported. Datasets: binary classification (1–2K samples, 10 features), regression (2K×10). Inference latency: single-sample predict over 10K iterations (p50/p99). Charts generated by scry-chart (dogfooding!).
